# 通信工学 インタラクティブ学習システム

通信工学の基本概念（サンプリング定理、パルス整形、AWGN）を視覚的・インタラクティブに学習するためのシステムです。

## セットアップ

### 1. 依存ライブラリのインストール

```bash
pip install -r requirements.txt
```

## 起動方法

### 🚀 もっとも簡単なローカル起動 (Windows)
新しく作成した `run.bat` を**ダブルクリック**するだけで、自動的にブラウザが立ち上がりアプリが起動します。
コマンド入力を省略できるので非常に便利です！

### コマンドラインからの起動 (従来の起動方法)

```bash
streamlit run app.py
```

## 🌐 GitHub連携 と Web上での公開

このプロジェクトはローカルだけでなく、GitHubで管理し、**Streamlit Community Cloud** を使えば無料で全世界に公開できます！これにより、スマホや他のPCからでも URL にアクセスするだけでアプリを使えるようになります。

### 公開手順 (Streamlit Community Cloud)

1. **GitHubにリポジトリを作成し、コードをプッシュする**
   このフォルダは既にGitで初期化されています（この後システム側で初期化を行います）。GitHubに新しいリポジトリを作り、プッシュしてください。
2. **[Streamlit Community Cloud](https://share.streamlit.io/) にログインする**
   （GitHubアカウントで無料でログインできます）
3. **「New app」をクリックし、リポジトリを連携する**
   作成したGitHubリポジトリ、ブランチ（`main`など）、メインのファイルパス（`app.py`）を選択し、「Deploy」をクリックするだけです。
4. 数分で **自分だけの専用URL**（例: `https://your-app-name.streamlit.app`）が発行され、いつでもどこでもアプリを使えるようになります！

## 学習の進め方

### 方法 A: 解説スクリプトで段階的に学ぶ（推奨：初めての方）

`tutorials/` フォルダ内のスクリプトを番号順に実行してください。

```bash
cd tutorials
python 01_sampling.py
python 02_pulse_shaping.py
python 03_awgn.py
```

各スクリプトには以下が含まれています:
- 概念の丁寧な解説（日本語コメント）
- 段階的なグラフ表示
- 「★ 実験してみよう」セクション（パラメータを変えて試す部分）
- セクションごとのまとめ

### 方法 B: Streamlit アプリでインタラクティブに操作

`streamlit run app.py` でアプリを起動し、スライダーを動かしてリアルタイムに変化を確認します。

## 学習内容

### 1. サンプリング定理とエイリアシング
- 連続信号の離散化（標本化）
- ナイキストレート（Nyquist rate）
- エイリアシングの発生メカニズム（時間領域・周波数領域）
- sinc 補間による信号の完全再構成

### 2. パルス整形とアイパターン
- 矩形パルスの問題点（帯域無限大）
- ナイキストの第1条件（ISI フリー条件）
- Raised Cosine (RC) フィルタとロールオフ率 α
- Root Raised Cosine (RRC) フィルタとマッチドフィルタ
- アイパターンによる信号品質評価

### 3. AWGN と変調方式
- AWGN の3つの性質（加法性・白色・ガウス性）
- 連続領域での波形劣化
- 離散領域でのコンスタレーション劣化
- 各変調方式の比較（BPSK, QPSK, 8PSK, 16QAM, 64QAM）
- BER（ビット誤り率）の理論値 vs シミュレーション
